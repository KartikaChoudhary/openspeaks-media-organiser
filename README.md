## What is this?

When people go into the field to document endangered and tribal languages, 
they come back with hundreds of files named things like `rec_final2.mp3` 
or `IMG_4821.jpg`. Organising them manually takes days.

This tool lets you:
- Open a folder of fieldwork recordings
- Preview each audio, video, or image file
- Add tags, notes, and categories per file
- Rename everything in bulk using a proper naming pattern
- Export a log of all renames as CSV

## Who is this for?

Built specifically for the [OpenSpeaks Archives](https://meta.wikimedia.org/wiki/OpenSpeaks/Tools) 
community who document indigenous and tribal languages across India and globally.

## How to use

1. Download `organiser.html`
2. Open it in Chrome or Edge
3. Click "Open Folder" and select your fieldwork files
4. Tag, preview, and rename away

No internet needed. No installation. No account. Just open and use.

## Naming pattern

Files get renamed using this structure:

`{language}-{type}-{names}-{subject}_{sequence}.{ext}`

Example: `gondi-audio-ravi-harvest_001.mp3`
